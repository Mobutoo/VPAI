## Critical Integration Gaps

### GAP 1 — BLOCKER: 4 NocoDB CRUD Webhooks Not Implemented

Every CF workflow calls `cf-create-content`, `cf-update-content`, `cf-read-content`, and `cf-scene` — but no workflow implements these webhook paths. These are the data layer glue between the content-director skill and the pipeline workflows.

**Impact:** The entire pipeline is dead at step 1 — `/content` calls `cf-create-content` which returns 404.

**Fix:** Create 4 n8n workflows that proxy CRUD operations to NocoDB API, register in Ansible loops.

### GAP 2 — BLOCKER: Ansible Variable Mismatch in Kitsu Event Handler

`event_handler.py.j2` references `n8n_webhook_hmac_secret` but inventory defines `n8n_webhook_secret`. Empty secret breaks all Kitsu → n8n events.

**Fix:** Change variable name in template to `n8n_webhook_secret`.

### GAP 3 — BLOCKER: Remotion API Endpoint/Field Mismatches

`creative-pipeline.json.j2` calls `/render` (should be `/renders`), sends `composition` (should be `compositionId`), expects `renderId` (server returns `jobId`).

**Fix:** Update creative-pipeline.json.j2 with correct endpoint, field names, and response handling.
