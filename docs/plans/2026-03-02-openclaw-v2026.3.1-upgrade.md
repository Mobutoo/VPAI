## Task 5: Update docker-compose.yml.j2 — HTTP healthcheck

```yaml
    # OpenClaw Gateway healthcheck: verify WebSocket upgrade (101) — not just HTTP 200 (SPA fallback)
    # HTTP GET always returns 200/index.html even when Gateway is in restart loop.
    # Only a successful WebSocket upgrade proves the Gateway is truly accepting connections.
    healthcheck:
      test: ["CMD-SHELL", "node -e \"const h=require('http');const r=h.request({host:'127.0.0.1',port:{{ openclaw_gateway_port }},path:'/',headers:{'Connection':'Upgrade','Upgrade':'websocket','Sec-WebSocket-Key':'dGhlIHNhbXBsZSBub25jZQ==','Sec-WebSocket-Version':'13'}});r.on('upgrade',(res)=>{process.exit(res.statusCode===101?0:1)});r.on('response',()=>process.exit(1));r.on('error',()=>process.exit(1));r.setTimeout(5000,()=>{r.destroy();process.exit(1)});r.end()\""]
      interval: 30s
      timeout: 10s
      retries: 5
      start_period: 60s
```

Replace with:

```yaml
    # OpenClaw Gateway healthcheck: native HTTP /healthz endpoint (v2026.3.1+ #31272)

    healthcheck:
      test: ["CMD-SHELL", "wget -q --spider http://127.0.0.1:{{ openclaw_gateway_port }}/healthz || exit 1"]
      interval: 30s
      timeout: 10s
      retries: 5
      start_period: 60s
```

```bash

import json, urllib.request, uuid, hashlib

QDRANT_URL = "http://qdrant:6333"
COLLECTION = "content_index"
OPENAI_URL = "http://litellm:4000/v1/embeddings"

    text = f.read()

    "input": text[:8000]

# Upsert into Qdrant
point = json.dumps({

        "id": doc_id,
        "vector": vector,
        "payload": {
            "source": "openclaw-release-notes",
            "version": "v2026.3.1",
            "date": "2026-03-02",
            "type": "release-notes",
            "text": text
        }
    }]
}).encode()
req = urllib.request.Request(f"{QDRANT_URL}/collections/{COLLECTION}/points", data=point, headers={
    "Content-Type": "application/json",
    "api-key": qdrant_key
}, method="PUT")
resp = urllib.request.urlopen(req, timeout=30)

PYEOF'
```
