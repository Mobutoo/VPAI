1.0  Objet — Perte NNI 16 carte LB

Ce MOP décrit la procédure complète de traitement d'une alarme de perte de port NNI 16 sur un châssis Ribbon OPTera Metro 5200. L'incident se traduit par une perte de trafic client sur un ou plusieurs circuits OTN. Symptômes Lightsoft : alarme LOS ou LOF sur port NNI, OSNR dégradé. Le technicien doit identifier si la cause est la carte LB, le SFP ou la jarretière optique avant toute intervention physique.

1.1  Symptômes observés

Alarme Lightsoft : LOS / LOF sur port NNI 16. Vizee : alerte orange/rouge sur l'équipement concerné. OSNR < 10 dB ou nul. Trafic client interrompu sur les circuits passant par ce port.

1.1.1  Alarmes associées

2.0  Rôles et responsabilités

NOC on-call : qualification initiale, exécution SOP-01. Ops Senior : validation diagnostic, décision bascule OLP. Ribbon TAC : escalade si remplacement carte nécessaire (SOP-10). Manager NOC : à informer dès criticité MAJEUR confirmée.

2.1  Matrice décision

SFP mort → SOP-07 (remplacement SFP). Carte LB défectueuse → SOP-10 (RMA Ribbon). Jarretière → intervention terrain Lugos. OLP basculé → SOP-04 (revert après réparation).

2.1.1  Critères d'escalade

3.0  Procédure d'intervention

3.1 Phase 1 — Qualification (SOP-01)


1. Ouvrir Lightsoft + Vizee simultanément.
2. Identifier l'alarme : LOS NNI-16, noter heure et châssis.
3. Vérifier si alarme isolée ou cascade.
4. Horodater. Criticité initiale : MAJEUR.

3.2 Phase 2 — Diagnostic (SOP-04 + SOP-07)


1. SOP-04 : vérifier bascule OLP active sur ce lien.
2. SOP-07 : mesurer puissance optique sur port NNI-16.
3. Si puissance nulle → SFP défectueux → remplacer.
4. Si puissance OK mais OSNR dégradé → jarretière ou carte LB.

3.3 Phase 3 — Escalade (SOP-10 si nécessaire)


1. Ouvrir SR Ribbon avec numéro de châssis + version SW.
2. Initier RMA si carte LB confirmée défectueuse.

3.4 Phase 4 — Communication (SOP-11)


1. Informer les sites impactés dès qualification.
2. Mail de suivi toutes les 30 min jusqu'à clôture.

3.2  Points d'attention critiques

⚠ NE bleu/gris ≠ perte de trafic — ne pas escalader avant confirmation OSNR.
⚠ Chassis 9603 = pas de doublon RCP — bascule manuelle impossible.
⚠ Toujours photographier l'état avant intervention physique.
⚠ Ne jamais extraire une carte sans SR Ribbon ouvert.

4.0  Glossaire & REX terrain

NNI : Network-to-Network Interface — port de raccordement inter-opérateurs.
LB : Line Board — carte de ligne OTN châssis Ribbon.
OLP : Optical Line Protection — protection automatique par bascule.
OSNR : Optical Signal-to-Noise Ratio — seuil critique : 12 dB.
SFP : Small Form-factor Pluggable — module optique remplaçable à chaud.

REX terrain Q1 2026 : 3 incidents similaires. Cause principale : SFP dégradé par poussière (2/3). Cause secondaire : jarretière mal connectée côté opérateur (1/3). Temps moyen résolution : 45 min.
