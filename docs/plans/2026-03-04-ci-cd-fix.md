### Task B3: Increase DNS propagation wait + add retry logic

**Files:**
- Modify: `.github/workflows/integration.yml` — job `provision`, step "Wait for DNS propagation"

**Step 1: Increase wait to 90s + test ALL subdomains**

```bash
      - name: Wait for DNS propagation (90s + validation)
        run: |
          echo "Waiting 90s for DNS propagation..."
          sleep 90
          SERVER_IP="${{ steps.server.outputs.server_ip }}"
          PREPROD_DOMAIN="${{ secrets.PREPROD_DOMAIN }}"
          PASS=true

          for sub in "" "n8n" "llm" "tala" "qdrant" "nocodb" "oc" "palais" "plane"; do
            if [ -n "${sub}" ]; then
              FQDN="${sub}.${PREPROD_DOMAIN}"
            else
              FQDN="${PREPROD_DOMAIN}"
            fi
            RESOLVED=$(dig +short "${FQDN}" @1.1.1.1 | head -1) || RESOLVED=""
            if [ "${RESOLVED}" = "${SERVER_IP}" ]; then
              echo "DNS OK: ${FQDN} -> ${RESOLVED}"
            else
              echo "WARN: ${FQDN} not resolved (got: '${RESOLVED}', expected: '${SERVER_IP}')"
              PASS=false
            fi
          done

          if [ "${PASS}" = "false" ]; then
            echo "Some DNS records not yet propagated — waiting 60s more..."
            sleep 60
          fi
```

        run: |
          D="${PREPROD_DOMAIN}"
          echo "Triggering ACME for all subdomains..."
          for sub in "" "n8n" "llm" "tala" "qdrant" "nocodb" "oc" "palais" "plane"; do
